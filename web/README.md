# SuperInbox Web / SuperInbox 前端应用

SuperInbox Web is the main user-facing interface for inbox browsing, review, and management.  
SuperInbox Web 是 SuperInbox 的主用户界面，用于收件箱浏览、处理与管理。

## Tech Stack / 技术栈

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- TanStack Query / Table

## Requirements / 运行要求

- Node.js >= 18
- npm

## Quick Start / 快速开始

```bash
npm install
npm run dev
```

Default URL / 默认访问地址: `http://localhost:3000`

## Build & Run / 构建与启动

```bash
npm run build
npm run start
```

## Lint / 代码检查

```bash
npm run lint
```

## Notes / 说明

- The web app is usually started together with backend via project root `./start.sh`.
- 建议通过项目根目录 `./start.sh` 与后端一起启动，减少端口和联调配置问题。
